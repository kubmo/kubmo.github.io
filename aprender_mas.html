Code and Design Resources

Recursos para aprender programación

Si usted está interesado en aprender más y seguir mejorando sus habilidades de desarrollo, hemos compilado una lista de recursos. Puedes hablar con su instructora sobre sus metas y ella puede hacer recomendaciones, también.

  • Codecademy: Un sitio en línea tutorial lenguaje que es de uso gratuito y en espanol.
  • 'Learn to Program'Un libro de Chris Pine.
  • La Laborotoria Escuela de Programacion en Lima. Andrea habla con ustedes con esa escuela de cuatro meses.
  • Railsbridge: Introducción a HTML and CSS. Es en ingles.

Recursos para aprender el diseño

Con el fin de crear una buena experiencia para los usuarios que utilizan sus sitios web o productos, es importante para entender el diseño. Hay un montón de recursos gratuitos para utilizar como guía e inspiración en la construcción de nuevas experiencias.

  • Linda: un sitio tutoría para aprender nuevas habilidades del diseño.
  • 'The rules of simplicity':un libro de principales del diseño. Es en ingles.
  • Google Design Documentation: Se aprende sobre las reglas de 'material design', un tipe de diseño.
  • Dribbble: un sitio donde los diseñadores mostran su trabajo y habilidad.
  • Google Ventures Design Sprints: Como se hace un prototipo rapidamente y se hace decisiones.
  • D. School Design Thinking: documentación del Stanford Design School
  • Google Fonts: La tecnica para encontrat fuentes gratis para usar directamente en su sitio de web, or para sus diseños.
  • Sketch: La tecnica para diseñar interfaz de los usadores. Puedes usar para diseñar sitios de web, applicaciones móviles, o cualquier cosa digital.
  • iOS UI Guidelines: Aprende sobre las pautas de Apple para crear aplicaciones de iOS.

Vocabulario Básico

  • HTML: 'Hyper Text Mark Up Language' Este es la idioma utilizado para construir la estructura de los sitios web, es como el marco de un edificio
  • CSS: 'Cascadin Style Sheets' Se utiliza para añadir estilo a sitios web. Ejemplos son la adición de color, fuentes personalizadas, o el espaciamiento
  • Elemento: Los elementos se utilizan para indicar al HTML qué hacer. Por ejemplo, el <h2> elemento dice la página que queremos un cabezazo, pero un cabezazo que es ligeramente menor que la cabecera mayor. Usted puede ver una lista de todas las etiquetas HTML aquí. También puede utilizar estes elementos de elementos en CSS cuando desea agregar estilo específico para ellos..
  • Los elementos de texto: Puede usar cosas como 'h1' para los encabezados, 'p' para el párrafo, y 'a' para los enlaces para ayudar estilo de su sitio con HTML
  • Browser: Un navegador es lo que se utiliza para conectarse a Internet. Es el lugar donde se puede ver su sitio web a nivel local, como lo construyes. Es también donde usted busca cosas en línea. Nuestro navegador favorito a utilizar es Google Chrome
  • Local: Cómo hacer referencia al código que está en el equipo y que está editando, pero aún no está vivo en editor dede
  • El editor de texto: Un editor de texto es donde se escribe y realizar ediciones a su código. Hay muchos tipos diferentes de editores de texto que puede utilizar, nuestro editor gratuito favorito es Sublime
  • "id": Un identificador se utiliza para apuntar elementos específicos. Usted debe utilizar esta opción cuando quieres hacer algo específico para un elemento, como el centrado sólo una imagen en la página
  • "class": Un identificador que se utiliza para dirigirse a un grupo de elementos. Usted debe utilizar esta opción cuando quieres hacer algo específico para un grupo de elementos, y reutilizarla. Por ejemplo, usted podría poner una clase en todos sus párrafos tener espaciamiento específico
  • Contenedor/'Container': El área donde su contenido vive. Puede cambiar la anchura, la altura, la alineación, el espaciado y más agregando estilos para la totalidad de su contenido, envuelto en un recipiente
  • Linea de altura/'Line height': La separación vertical entre líneas de texto. El espacio correcto entre líneas de texto hace que sea más fácil de leer
  • Margin/Margen: La distancia que desea un elemento a tener de otros elementos. Por ejemplo, usted podría dar el título de un margen izquierdo de 20 píxeles para que se mueva 20px a la derecha
  • Relleno/'Padding': The distance of space you want to have around an element. For example, you could put 20px padding around an entire piece of text, and add color to it to create a button.
  • Encabezado/'Header': La información que siempre se sienta en la parte superior de una página web. Esta área por lo general incluye la navegación, un logotipo, y las cosas que usted desea que el usuario haga clic enpágina
  • Pie/'Footer': La información que siempre se sienta en la parte inferior de una página web. Esta área puede incluir enlaces importantes, información o campos de entrada para las personas a encontrar fácilmente la información adicional.
